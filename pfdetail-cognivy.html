Cognivy : Alzheimer Dimenthia Theraphy

Project information

  • Platform: iOS
  • Project date: Juli, 2024
  • Project URL: -

An AR app that help early-stage Alzheimer's patients with memory problem to do cognitive memory therapy by placing virtual items in specific locations with relevan context and reordering them after randomization.

Features:
- AR Interaction : uses AR to place virtual items in real-world environments, providing an interactive experience for memory therapy, Patients perform cognitive exercises by reordering virtual objects, helping to stimulate memory recall and cognitive function, Objects are randomly shuffled, and patients are tasked with reordering them to their original locations, improving recall skills. - Theraphy Tracking Progress : Tracks their progress over time each session, adapting to their abilities for better therapy outcomes.
