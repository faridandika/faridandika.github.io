Capstone : Gym Guide

Project Information

  • Output: Android App
  • Project Date: December 2023
  • Project URL: https://github.com/GymGuide

Gym Guide is an Android app that can guide user to exercise with or without gym equipment,it can scans photos of gym equipment, processes them using CNN, and generates tutorials on how to use that equipment. Our application focuses on introverted personalities, newcomers, individuals who are unsure how to properly operate gym equipment. Helping them to operate gym equipment correctly, eliminating the necessity to seek help from others.

Features:

  • Login & Register: with Google Mail and Facebook API that helps user easily logging in and instant access.
  • Scan Gym Equipment: using CNN it can generates tutorials and guidance on how to use the gym equipment.
  • Consultation: with Google Maps API able to saw the closer trainers around which is also another user of the GymGuide app.
  • Muscle Map: tap specific muscle of human body image that you want to improve and simply received the exercise recomendation with the guidance inside of it.
